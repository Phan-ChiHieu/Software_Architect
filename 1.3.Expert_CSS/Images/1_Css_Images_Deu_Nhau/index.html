Hình ảnh không đều nhau




Dùng CSS để Hình ảnh đều nhau

Vào css xem class .gallery-panel




Dùng aspect-ratio để căn chỉnh width height của hình ảnh




aspect-ratio là một thuộc tính CSS dùng để xác định tỷ lệ chiều rộng so với chiều cao của một phần tử. Khi bạn đặt thuộc tính này, trình duyệt sẽ tự động duy trì tỷ lệ giữa chiều rộng và chiều cao của phần tử bất kể kích thước cụ thể của phần tử đó có thay đổi như thế nào.
